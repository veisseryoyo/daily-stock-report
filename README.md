📜 יומן המסע: בניית בוט המניות של יהונתן
🚀 המטרה
יצירת מערכת אוטומטית שרצה פעם ביום, מושכת נתוני מסחר של מניית AT&T (סימול: T) ושולחת דוח מעוצב הכולל מחיר, אחוז שינוי ושווי תיק אישי (24 מניות).

📉 הנפילות (האתגרים שבדרך)
מחסום ה-Gmail: התחלנו עם ניסיון לשלוח מיילים דרך שרת ה-SMTP של גוגל. גוגל חסמה אותנו שוב ושוב בטענה של "Bad Credentials", למרות השימוש ב-App Password. הבנו שגוגל מקשיחה עמדות מול שרתים של GitHub.

נפילת שירותי המייל החיצוניים: ניסינו לעבור ל-SendGrid ו-Brevo. ב-SendGrid החשבון נחסם מיד, וב-Brevo המיילים פשוט לא הגיעו כי הם דרשו אימות דומיין מורכב.

תקלות ה-YAML: נלחמנו בשגיאות סינטקס בקובץ ה-main.yml (שורות רווח, טאבים ומשתני סביבה חסרים) שמנעו מהסקריפט לרוץ בכלל.

שגיאת "משיכת הנתונים": גם כשהסקריפט כבר רץ, קיבלנו שגיאות כי ה-API של Finnhub לא קיבל את המפתח (Secret) בצורה נכונה מה-GitHub Actions.

📈 העליות והרגעים הגדולים
המעבר ל-Discord: הרגע שבו החלטנו "לשבור את הכלים" ולעבור מאימייל ל-Webhook של דיסקורד. זה היה המפנה שהסיר את כל חסמי האבטחה המעיקים.

פיצוח ה-Secrets: הצלחנו להגדיר נכון את ה-DISCORD_WEBHOOK וה-FINNHUB_KEY בתוך המערכת של GitHub, כך שהקוד נשאר מאובטח אבל מתפקד.

גמישות המקורות: הוספנו מנגנון "אל-כשל" – אם ה-API של Finnhub נכשל, הסקריפט יודע ללכת ל-Yahoo Finance כדי להביא את הנתונים, מה שמבטיח שהבוט לא ימות לעולם.

החזון לעתיד: ההחלטה להפוך את זה בעתיד לבוט קהילתי שכולם יוכלו להיעזר בו.

🛠️ המצב הנוכחי (מה עובד עכשיו)
קוד: פייתון יציב שרץ על GitHub Actions.

לו"ז: הבוט רץ אוטומטית כל יום בשעה 23:30 (שעון ישראל).

תוצאה: הודעה מעוצבת בדיסקורד עם אימוג'ים, מחיר עדכני וחישוב רווח/הפסד יומי.

📝 הוראות למקרה של "שבירת כלים" בעתיד:
מפתחות: תמיד לוודא ששמות ה-Secrets ב-GitHub תואמים בדיוק למילים FINNHUB_KEY ו-DISCORD_WEBHOOK.

שינוי מניות: כדי להוסיף מניות נוספות, פשוט מעדכנים את רשימת ה-WATCHLIST בקוד.
